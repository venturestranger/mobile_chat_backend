### Запуск сервера
1. Открыть консоль
2. Зайти в папку с бэкэндом
3. Выполнить программу 'pipX.X install -r requirements.txt' - чтобы установить все библиотеки для сервера (pipX.X - твоя установленная версия pip)
4. Выполнить программу 'pythonX.X main.py' - чтобы включить сервер (pythonX.X - твоя установленная версия python)

### Инициализация БД
5. Если хочешь заново инициализировать базу данных, зайди в файл config.py и найди поле DB_INIT - задай его True, чтобы инициализировать заново базу данных. Запусти сервер, потом выключи. После чего, поставь этот флаг в False обратно, чтобы он не создавал заново БД. Можешь заново запускать сервер и он будет работать с обновленной базой данных

### Установление соединения клиента с сервером
6. Когда будешь пользоваться клиентом, он будет запрашивать IP-address. Нужно чтобы клиент был подключен в ту же локальную сеть, что и backend сервер. В этой локальной сети нужно узнать IP-address сервера и в самом уже клиенте вписать 'IP-address:7890', где 7890 - порт для сервера
7. Если подключаться не будет, то надо отключить брэндмауэр виндоус на время. С остальными ОС проблем не должно возникать
